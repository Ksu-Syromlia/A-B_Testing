# A-B_Testing
Discount Badge Impact on Subscription Conversion

**Objective**: Designed and executed a rigorous A/B test to evaluate the impact of a "50% Off" anchor-pricing badge on onboarding-to-subscription conversion rates for a new mobile application.

**Methodology**:  Conducted a 23-day experiment with ~20,000 unique users, ensuring adequate sample size to account for weekly seasonality.Validated results using multi-method statistical analysis (Z-test, Chi-squared, and Permutation tests), ensuring p < 0.05 significance.
